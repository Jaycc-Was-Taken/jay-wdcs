Colors

Using a color palette is important to me because I like to make sure that all of the colors I want to use are complimentary. I generally make my palettes 5 colors total, and personally, I like to pick a shade of green to use as the base of my palette since it is my favorite color, I'm a fan of pretty much any shade of green!

Fonts

The font this website uses is Garamond, it is a serif typeface created by sixteenth-century Parisian engraver Claude Garamond. It is a CSS Websafe Font that is generally used for printed books, but I like it for its readability.

A CSS Websafe Font is one that is available natively through HTML/CSS for use on webpages. It is generally a good idea to choose a fallback font that is one of the CSS Websafe Fonts in case an imported font does not work for the users browser. Even though HTML includes the <em> and <strong> to italicize and bold text, you will still need to include the bold and italic files if you are including the font in the websites files, and not just linking it from Google Fonts. In these situations, it is highly recommended to use a fallback font in case the included font does not load for the user.

Fonts can also have different sizes for the same font size. Make sure you take that into account when designing the site and picking the font family and fallback fonts. The font size of the fonts below are all 18px in size.
