Semantic Elements

A Semantic Element clearly defines it's meaning.
Tag Summary
<header> This represents a container for intro content, or nav links.
<article> Specifies independent, self-contained content.
<section> Defines a section of a document.
<aside> Defines content aside from parent content, should be indirectly related.
<details> Defines additional details. User can choose to show/hide.
<summary> Defines a visible heading for <details> element.
<figure> Specifies self-contained content. Generally used for illustrations, diagrams, photos, code listings, etc.
<figcaption> Caption for the <figure> element.
<table> Defines a table.
<main> Specifies main content of a document.
<mark> Highlights/marks text.
<nav> Navigation links.
<time> Defines the date and time.
<footer> Marks the footer of document or section.
