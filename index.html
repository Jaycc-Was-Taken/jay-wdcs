I am Jay and I am currently taking the Full-Stack Engineer career path on Codecademy

I am a big fan of being able to fully use a website when you have two different windows right next to each other on one monitor. So this website is primarily designed with that in mind, so that I am able to code and reference the site easily on one 1080p monitor, with VSC being larger than the website.

A lot of this site will feature design elements that were a lot more common in earlier days of the internet and early web2.0, but I really like them.
